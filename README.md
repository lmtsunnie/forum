

# "北理人"校内论坛项目
## 功能

登录、注册、发送激活邮件、修改个人信息、发帖、给帖子点赞、回复帖子、关注取关用户等

## 开发环境

操作系统：macOS Mojave 10.14.2  

IDE： IntelliJ idea

## 技术栈  

构建工具：maven  

框架：SpringMVC  

持久层：MyBatis  

数据库：MySQL / Redis  

前端：jQuery  

## 程序运行操作步骤

1. 安装JDK 1.8
2. 安装tomcat服务器
3. 安装并运行redis服务器
4. 将程序文件用idea打开，下载maven依赖
5. 安装mysql，将df.sql文件中内容写入数据库建库建表
6. 配置tomcat服务器 
7. 运行程序
